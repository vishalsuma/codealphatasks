# codealphatasks# CodeAlpha – Low Fidelity UX Design  
## Food Delivery App

This repository contains the **low-fidelity UX prototype** for a Food Delivery mobile/web app.

The purpose of this project is to demonstrate a structured user flow and functional layout using low-fidelity wireframes.

---

## 🚀 Project Overview

Low-fidelity UX design focuses on:

- Structure & layout
- Navigation flow
- Content placement
- Simplicity over visuals

This design was created for the **Food Delivery** user experience.

Screenshots have been exported from Figma.

## 🧠 User Flow

1. **Home**
   - Main navigation
   - Categories
   - Top Restaurants

2. **Browse Restaurants**
   - Filter by category
   - Scroll list

3. **Menu**
   - Add items to cart
   - View prices

4. **Cart / Checkout**
   - List of selected items
   - Subtotal & fees
   - Checkout button
